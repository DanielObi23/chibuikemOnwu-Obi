Rules for input

  • Must contain at least one alphanumeric character (letters or numbers).
  • Can include spaces, commas, hyphens, and periods for formatting.
  • Must not contain special characters like @, #, $, %, etc., unless part of a valid address format (e.g., unit numbers like #123).
  • Must not exceed 100 characters in length.
  • Must include at least one street name or landmark.
  • Optional: Can include a city, state, or postal code for better accuracy.
  • Must not be empty or consist solely of whitespace.

e.g Frauenplan 1, 99423 Weimar, Germany
